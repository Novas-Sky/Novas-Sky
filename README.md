# 群星计划

## 看见彼此，然后改变世界。

这个计划。叫做群星计划。

网上创业的00后，90后等的榜样，关于如何建立事业，实现价值的故事，已经很多了。这些故事，这些人，被看见，被学习。

但是向更年轻的角落看，青少年的故事很少，自己搞技术的很少，即使有，我们也很难看见，很难知道。

但是他们肯定存在。

那很多想去创造的青少年，不被看见，被家庭所阻止的青少年又该怎么去做？他们可能有着那颗想去做的心，但他们很难看到与自己一样的人在做，很难看见同路人。他们可能不敢去做，可能不知道怎么去做

就像是还没有被点亮的星。

所以群星计划想提供的就是点亮这颗星的一点动力。

不是物质上的东西，至少目前不是，我们目前没有这个能力，也并不想这样做。而是精神上的东西，群星计划想提供一些故事，一些方法，一些思考。

这些东西来自可能一样不被支持，但已经在走的人。也可能来自已经走过去，愿意提供自己的故事和方法的人。

### 仓库组成：

#### 一、故事（stories）：

在这里，你可以讲述你自己的关于创造的故事，无论是现在的，还是之前的。你的故事能激励到其他和你一样的人去做；同样的，你也许现在不敢去做，去创造。你可以在里面看到与你一样的人在创造，如果你能因这些同路人的故事敢去做一点点，去争取一点点，那群星计划的目的便达到了。

#### 二、方法(approaches)：
如果你现在或曾经成功的争取到了一点自由，找到了学习或现实东西与热爱的平衡，请写下来，帮助同路人；你也能看到其他人的方法，被其他人所帮助。

#### 三、交流(connect)：
你可以发出你的疑问和思考，同样的，你也可以回答他人的疑问与思考。这里不评判，只是互相吸引，汇聚。

我是一个初二的学生，没有耀眼的成就，不是什么榜样，我只是一个同样的，不被家庭支持，没有什么资源，但正在努力做的人。群星计划正是源于这真实的困境产生的，于是我启动了群星计划。

让每一个在现实的困境中的创造者看见彼此。

然后，开始做一点点，争取到一点点。

让星辰同在，并熠熠生辉。

## 我们相信，群星终将闪耀。
